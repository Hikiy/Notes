# 安装Node.js和npm
- [Node.js官网下载](https://nodejs.org)
- cmd输入`node -v`查看版本
- cmd输入`npm -v`查看版本（npm会附属下载）

io.js是“尝鲜版”，而Node.js是线上稳定版
