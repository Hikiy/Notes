# nvm 对node.js进行版本控制
### 安装
命令行：  
- 安装当前最新的稳定版：
```
nvm install stable
```
- 安装指定版本（PS:Windows 10 用不了8.16.0版本，还不知道原因）：
```
nvm install 8.16.0
```

### 更换Node版本
```
nvm use 8.16.0
```
### 显示所有安装的Node版本
```
nvm ls
```
