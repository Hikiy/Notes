# Eclipse properties 自动转码问题
因为eclipse中 properties文件默认编码类型为ISO，要改为utf8  
Eclipse->window->Preference  
搜索Content Types  
找到Text->Java Properties File，将Default encoding改为utf8（PS：改完后一定要点击Update）
